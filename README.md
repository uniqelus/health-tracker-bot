# TrackHub

Telegram Mini App для трекинга здоровья: питание, тренировки, прогресс тела.

## Стек

- **Backend:** Python, FastAPI, pytelegrambotapi
- **DB:** PostgreSQL (planned)
- **Infra:** Docker Compose, UV, Ruff

## Запуск

```bash
cp .env.example .env  # заполнить переменные
make up               # поднять всё через Docker
```
